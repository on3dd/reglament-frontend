## how to install

### clone this repository

```
# via http
$ git clone https://github.com/on3dd/reglament-frontend.git
```

or

```
# via ssh
$ git clone git@github.com:on3dd/reglament-frontend.git
```

### enter the project directory 

```
$ cd reglament-frontend
```

### install dependencies
```
npm install
```

## how to start work with app

go to created folder and run:
`yarn start` || `npm run start` ‚Äî this will start dev server with hot reload on `localhost:3000`.

`yarn build` || `npm run build` ‚Äî this will build production bundle, with tree-shaking, uglify and all this modern fancy stuff
